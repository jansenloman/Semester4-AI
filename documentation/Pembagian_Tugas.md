# Garisbesar

Pembagian tugas berikut adalah gambaran besar dari apa yang dikerjakan masing-masing anggota. Jika anda ingin melihat detailnya:

### Jika menggunakan Git
Ketikkan

```git log```

di Terminal anda jika anda clone repository ini dan gunakan tombol *Arrow Up* dan *Arrow Down* untuk navigasi log dari Git.

### Jika menggunakan GitHub
Klik tombol "[number] commits" dengan icon jam diatas list file dan folder atau ke URL "https://github.com/DavinTristanIeson/Semester4-AI/commits/main" untuk melihat semua commit.
